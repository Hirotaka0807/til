# Linuxとは
- Unixを元に開発された、オープンソースのOS。  

# シェルについて
カーネル（OSの核）の周りにあるもの（もの？）。  
カーネルには直接命令は出せないので、Shellにコマンドで命令を出し、ShellがKernelに命令を出す。  
bash, zsh, shなどが存在する。

# 環境変数
- cd <path>  
  Change directory
- pwd  
  print working directory
- mkdir  
  make new folder
- touch  
  make new file
- ls  
  list files in working directory
- rm  
  remove file or folder
